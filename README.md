# PobierakYT - Retro-style yt-dlp GUI
[![scr-Pobierak.png](https://github.com/CelularBat/PobierakYT/blob/main/readme/frontScreen.png)](https://github.com/CelularBat/PobierakYT/blob/main/readme/frontScreen.png)

### Features:
 - Raw and simple style like in the '90s !
 - Utilizes full potential of yt-dlp!
 - Flexible and customizable - you can use quick settings to download with one click OR you can define lists of your own custom settings and save them! 
 - Lightweight and portable, below <4 MB , needs NO dependencies
 - Capable of executing many tasks at the same time, asynchronously 
 [![scr-Pobierak.png](https://github.com/CelularBat/PobierakYT/blob/main/readme/advScreen.png)](https://github.com/CelularBat/PobierakYT/blob/main/readme/advScreen.png)
 You can define your own commands to use with yt-dlp, output templates and format quality strings.
### Functionality:
PobierakYT offers many advanced yt-dlp functions with just a click of a button:
 - Downloading piece of video between defined time frames
 - Downloading only some chapters or all chapters separated into individual files
 - Manually choosing audio/video quality from the list
 - Downloading subtitles

## Requiments
PobierakYT is currently only available for Windows (7 / 8 /8.1 /10 / 11) ,but Linux support is also planned.
 - You need to have yt-dlp.exe ( https://github.com/yt-dlp/yt-dlp?tab=readme-ov-file#installation )
 - You need to have folder with FFMPEG binaries for yt-dlp ( https://github.com/yt-dlp/FFmpeg-Builds#ffmpeg-static-auto-builds )
